# Running the application locally

Install pnpm

### `npm i -g pnpm`

Install the libraries

### `pnpm install`

Create the .env from .env.example

### `cp .env.example .env`

Run the application.

### `pnpm start`

> Note: Commands should be run at the root. Application should be running in `http://127.0.0.1:3000/`. **DO NOT USE LOCALHOST**
